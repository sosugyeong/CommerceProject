# 🛒 Commerce System
객체지향 커머스 플랫폼
- **언어** : Java
- **Tools** : IntelliJ, Git
- **기간** : 26.01.17 ~ 26.01.23

## ✔️ 주요기능
### 상품
- 전체상품 보기
- 100만원 이상 상품 필터링
- 100만원 이하 상품 필터링

### 카테고리
- 상품을 관리

### 고객
- 고객 정보를 관리
- 누적 주문금액에 따른 등급 결정

### 장바구니
- 장바구니에 담김 모든 상품, 총 금액 출력
- 장바구니 담기 기능
- 장바구니 주문 기능

### 주문
- 장바구니를 초기화하고 재고 차감

### 관리자모드
- 관리자 인증 기능
- 새로운 상품 추가
- 기존 상품 수정 / 삭제 / 전체 출력

### 고객 등급 할인
- Enum을 활용한 고객 등급별 할인율 관리
  - BRONZE(0%), SILVER(5%), GOLD(10%), PLATINUM(15%)
  - 등급에 맞는 할인율 적용해 총 금액 계산
